# Enchant Overlay PNGs

256×256px PNG files with dark background (not removed — eliminated by additive blending at render time).

Naming convention: `enchant_<id>.png`

| File | Enchant | Pattern | Colour |
|---|---|---|---|
| `enchant_gilded.png` | GILDED | Honeycomb hexagons | Gold `#d4a800` |
| `enchant_reverb.png` | REVERB / ECHOZZ | Rainbow rays from centre | Rainbow |
| `enchant_boomerang.png` | BOOMERANG | Holo waves | Teal `#1a8a3a` |
| `enchant_ironclad.png` | IRONCLAD | Maze / grid lines | Silver `#555566` |
| `enchant_feather.png` | FEATHER | Crackle / crazing | Light blue `#0099cc` |
| `enchant_halflife.png` | HALFLIFE | Circuit board traces | Green `#00cc88` |

Drop your test PNG in here as `enchant_test.png` to use as shared placeholder.
